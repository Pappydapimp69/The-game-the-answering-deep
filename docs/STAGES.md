# Build Stages

## Stage 0 — full first build

Game 4 of the saga, following The Waiting City. A lightless drowned reach past the harbor gate, where you perceive by ECHO and something has learned to answer.

Built by continuing the saga's established pattern (each game rebuilds its own determinism spine rather than sharing runtime code), scaffolded from the Waiting City so this session's mobile/deploy fixes came for free, then transformed.

**The signature system — ECHO (`src/sim/sound.js`, the new primitive):** the world renders dark beyond a small ambient radius; a `PING` command floods sound outward through open space (walls block conduction, so they bound both what you hear and where your noise carries) and briefly **lights** every tile it reaches (`state.echo.lit`, authoritative, pruned each TICK so it stays bounded and fingerprinted). The cost: every sound-reactive creature within earshot flips to a **two-stage alert** — first `search`, homing on *the tile the sound came from* (a last-known-position, so it arrives a beat late where the sound WAS — the Yamabiko), then de-escalating back to post if the sound turns up nothing, or escalating to `chase`/`attack` if it actually finds you. Silence is safety and blindness; sound is sight and exposure. A quiet ping is free and short; a loud pulse spends aura to see and be heard far. A ranged aura-blast requires a currently-lit target (you can't blast what you haven't echo-located); melee is exempt (range 1). Sound propagation is a pure integer BFS flood reusing the saga's deterministic-BFS idiom — no floating-point acoustics, no ambient time, covered by the golden-fingerprint replay.

**The theme — the "artificial threat" beat, delivered as sound:** the saga's own 5-part roadmap (home → rival → tyrant → **artificial threat** → apocalyptic) reserved this beat for game 4, and the Waiting City's finale set it up verbatim ("something answers that was never given a voice to do it with"). The Answerer is a deep-sea mimic (Yamabiko / mythic Echo / the Voice-Changeling): early it only repeats your pings a beat late; by half-health it stops waiting to be called and answers first — an intelligence acquiring agency through imitation, with zero sci-fi vocabulary. Mechanically it's a high-`hearing` creature that reliably picks up every pulse and homes on where you called from, so "it answers your call" IS the search-homing FSM.

**Continuity:** imports the Waiting City's `saga.v3` (SAGA3) — archetype, difficulty, skills, coins, and the accumulated choices (ravagerFate, riftChoice, wardenFate) — and exports `saga.v4` (SAGA4) adding this game's fate: `answererFate` = `silence` (end the stolen voice, +aura) or `answer` (give it a real one, +perception and the `deep-voice` technique). The whole choice chain accumulates to the finale (game 5).

**Signature reused, not repeated:** the perception→AI-legibility readout from game 3 carries over but now reads the deep's *alert* state (has it heard you — `unaware`/`heard you`/`losing you`), a genuinely new state to read rather than a relabel. The walk-behind-structure fade (game 3) and the deterministic ambient movers (drifters on currents, game 3's cars) are kept as the low-stakes proof of the movement machinery.

**Determinism & guards (all prior saga lessons pre-loaded):** canonical sorted-key serialization + seeded-RNG raw-state save + a CI grep-guard banning `Math.random`/`Date.now`/`performance.now`/`new Date`/transcendental `Math.*` in `src/sim`; the golden-fingerprint replay (`60716a35`); the validation ladder (schema → referential-integrity/completability → headless smoke); mechanically-frozen read-only renderer; event-time input capture; existence-gated quest targets + sim-side spawn telegraph; shop-NPC ≠ quest-giver; respawn-from-JSON-snapshot. Mobile/deploy from this session baked in at commit 1: `100dvh/100dvw` canvas sizing, `navigator.maxTouchPoints` touch detect, `Math.min(W,H)` world scale, and a deploy-time step that cache-busts **every** nested ES-module import (not just the entry).

**Verified before shipping:** `npm run smoke` (54/54, including dedicated echo/sound/alert coverage — BFS field shape, ping lighting + expiry, the reveal-gate on aura-blast, loud-vs-quiet reach/cost, heard→search-toward-origin, and searcher de-escalation) plus a browser e2e (boot with zero console errors, in-browser golden-fingerprint parity, and a live check that a quiet ping and then a loud pulse light progressively more of the dark).

**Grounding:** two Brain read fan-outs (`memory/PITFALLS.md`, `ideas/idea-repository.md`) and five external research passes (echolocation-as-sight games — Dark Echo/Stifled/Perception/Muffled Warfare; voice-mimic horror — Yamabiko/Mimic/Echo; grid noise radii — Invisible Inc; stealth two-stage-alert fairness; audio localizability).
